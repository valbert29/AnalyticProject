
                        Система "Исполнители"
                             версия 2.5
                             
                  (C) Константин Поляков, 2000-2007


Система "Исполнители" представляет собой интегрированную  оболочку для
начального обучения по теме "Алгоритмы и исполнители" в школьном курсе
информатики. Исполнители (Робот, Чертёжник и Черепаха) выполняют программу,
которая вводится в текстовом редакторе. Все действия исполнителя
отображаются на экране.

Главные достоинства программы
------------------------------
-  возможность составления и решения разных типов задач по темам
   "Исполнитель и его команды", "Процедуры", "Функции", "Циклы",
   "Условные операторы", "Переменные", "Арифметические выражения",
   "Логические операции и логические переменные", "Глобальные
   переменные", "Операторы ввода и вывода" и др.;
-  язык программирования полностью русифицирован;
   программы набираются и редактируются с помощью встроенного
   редактора с подсветкой синтаксиса;
-  поддержка базовых конструкций языка Си;
-  диагностика различных видов ошибок при трансляции и
   выполнении;
-  встроенный отладчик с возможность выполнения программ в
   пошаговом режиме с трассировкой процедур и функций;
   возможность контроля значений переменных во время
   выполнения в пошаговом режиме;
-  анимация при выполнении программы;
-  проверка правильности решения задачи после окончания работы
   программы;

Что нового в версии 2.1?
-------------------------
Версия 2.1 системы "Исполнители" включает следующие изменения в
сравнении с версией 2.0: 
-  Редактор программы теперь работает с горизонтальной полосой прокрутки.
   В редакторе программы можно выделить слово двойным щелчком по нему.
   При нажатии на клавишу <b>Delete</b> в редакторе выделенный фрагмент 
   удаляется.
-  При загрузке новой задачи для Робота предусмотрен автоматический 
   выбор масштаба.
-  В язык программирования добавлена команда "очистить".
   для очистки рабочего экрана. При работе в среде для выполнения этой операции
   можно нажать кнопку F12.
-  В язык программирования добавлена возможность использовать
   операторы потокового ввода и вывода данных в современной форме языка Си.
-  Добавлена возможность сохранения лабиринта с другим именем 
   с помощью комбинации клавиш Shift-Ctrl-M или через верхее меню.

Что нового в версии 2.0?
--------------------------
Версия 2.0 системы "Исполнители" включает следующие изменения в
сравнении с версией 1.2: 

-  Добавлена поддержка одномерных и двухмерных массивов, 
   в том числе массивов-параметров в процедурах и функциях.
-  Добавлена поддержка символьных строк.
-  Если программа не обращается к исполнителю, его поле
   скрывается с экрана; чтобы снова вызвать окно исполнителя,
   надо щелкнуть на его кнопке на панели инструментов.
-  В главное меню добавлен раздел Шаблоны.
-  Команды Робота 'влево' и 'вправо' заменены на
   команды 'налево' и 'направо' соответственно.
-  Оператор вывода автоматически не вызывает переход на новую строку, чтобы
   перейти на новую строку в конце вывода используется оператор println
-  Добавлена возможность форматирования выводимой информации
   (указывается размер поля для элемента и количество цифр в дробной
   части для вещественных чисел).
-  Курсор редактора задач показывает, во что превратится
   клетка поля после щелчка мышью.
-  Начальное положение Робота может быть любым (Робота можно
   повернуть в любую сторону).
-  Во время пошагового выполнения программы можно просматривать значения 
   переменных и массивов, используя всплывающие подсказки,
   которые появляются при наведении мыши на имя переменной в окне 
   редактора программ.
-  Сделан интерфейс в стиле Microsoft Office XP.
-  Сделан экспорт в формат HTML с сохранением форматирования
   и цветового выделения синтаксических элементов.

Системные требования
----------------------
Программа работает под управлением 32-разрядных операционных систем
линейки Windows: Windows 95, Windows 98, Windows NT, Windows 2000 и
совместимых с ними. Требования к аппаратной части:

        -  процессор Pentium 100
        -  оперативная память не менее 8 Мб
        -  свободное место на жестком диске 1 Мб
        -  рекомендуемое разрешение экрана 800 на 600 пикселов или выше
        
Справочная система построена в виде сжатого гипертекста в формате
HTML Help (файл ROBOT.CHM). Для просмотра требуется установленный
бруазер Internet Explorer версии 4.0 или выше.

При наличии Internet Explorer версии 3.0 требуется скачать файл
HHUPD.EXE с сайта www.microsoft.com и запустить его для установки
программы HH.EXE и сопутствующих библиотек.
При этом вместо контекстных подсказок будет вызываться
справочная система в отдельном окне.
